Sobre Mim

Desde o início da minha jornada na tecnologia, sou movido pela curiosidade e pelo desafio de criar soluções que simplificam a vida das pessoas. Com experiência prática no desenvolvimento de projetos e automação de processos usando Python e SQL, aprendi a importância de um código limpo e eficiente.

Finalizando a graduação em Análise e Desenvolvimento de Sistemas, meu objetivo é aplicar a engenharia de software para construir sistemas inteligentes. Utilizo minhas habilidades em análise de dados para criar soluções que otimizam operações e geram insights valiosos para os negócios.
